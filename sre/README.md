# SRE Skills

Planned: 15 skills. 15 built so far.

Each skill lives in its own subfolder under [`skills/`](./skills) with a `SKILL.md` (instructions Claude follows when the skill runs) and a `README.md` (what it does, how to invoke it, and a worked example).

## Built

| Skill | What it does |
|---|---|
| [`slo-review`](./skills/slo-review) | Org-wide SLO coverage, consistency, and whether they actually drive decisions. |
| [`sla-review`](./skills/sla-review) | External SLA commitments vs. actual internal reliability capability. |
| [`error-budget-review`](./skills/error-budget-review) | Cross-team error budget policy consistency and enforcement follow-through. |
| [`reliability-assessment`](./skills/reliability-assessment) | Single-service holistic reliability picture and dominant-risk identification. |
| [`toil-detection`](./skills/toil-detection) | Identifies and prioritizes toil for automation by aggregate cost. |
| [`capacity-planning`](./skills/capacity-planning) | Projects SRE/on-call team capacity against portfolio growth. |
| [`reliability-roadmap`](./skills/reliability-roadmap) | Sequences reliability findings into an impact/effort-prioritized roadmap. |
| [`availability-review`](./skills/availability-review) | Actual availability computation, cause decomposition, and trend. |
| [`chaos-engineering-review`](./skills/chaos-engineering-review) | Hypothesis-driven design, blast-radius control, findings follow-through. |
| [`runbook-review`](./skills/runbook-review) | Coverage, findability, currency, and validation status across runbooks. |
| [`operational-readiness`](./skills/operational-readiness) | Pre-launch/ongoing go-no-go checklist across readiness dimensions. |
| [`service-health-review`](./skills/service-health-review) | Lightweight recurring health snapshot for ops review cadence. |
| [`incident-trend-review`](./skills/incident-trend-review) | Translates incident trends into capacity-aware investment priorities. |
| [`reliability-kpis`](./skills/reliability-kpis) | Reviews whether reliability metrics measure what matters and roll up sensibly. |
| [`service-maturity-assessment`](./skills/service-maturity-assessment) | Portfolio-wide maturity scoring weighted by service criticality. |
