# Productivity Skills

Planned: 20 skills. 20 built so far.

Each skill lives in its own subfolder under [`skills/`](./skills) with a `SKILL.md` (instructions Claude follows when the skill runs) and a `README.md` (what it does, how to invoke it, and a worked example).

## Built

| Skill | What it does |
|---|---|
| [`design-doc-review`](./skills/design-doc-review) | Clarity, alternatives, and honest tradeoff coverage in a design doc. |
| [`architecture-review`](./skills/architecture-review) | Generic architecture documentation clarity, consistency, and currency. |
| [`adr-generator`](./skills/adr-generator) | Generates an Architecture Decision Record from a decision context. |
| [`rfc-review`](./skills/rfc-review) | RFC process readiness, scope clarity, and genuine feedback invitation. |
| [`meeting-notes-summary`](./skills/meeting-notes-summary) | Decisions, owned action items, and open questions from raw notes. |
| [`technical-documentation`](./skills/technical-documentation) | General doc review/writing for clarity, structure, and audience fit. |
| [`api-documentation`](./skills/api-documentation) | Endpoint/parameter completeness and example accuracy for API docs. |
| [`readme-generator`](./skills/readme-generator) | Generates/reviews a project README tailored to its actual audience. |
| [`migration-plan`](./skills/migration-plan) | Phasing, rollback points, and stakeholder coordination for a migration. |
| [`refactoring-plan`](./skills/refactoring-plan) | Scope, test-coverage adequacy, and safe incremental sequencing. |
| [`codebase-onboarding`](./skills/codebase-onboarding) | Core concepts, reading order, and a first task for a new engineer. |
| [`sprint-planning`](./skills/sprint-planning) | Backlog readiness and capacity sizing grounded in real velocity. |
| [`technical-debt-analysis`](./skills/technical-debt-analysis) | Distinguishes costly debt from cosmetic debt, quantified by evidence. |
| [`dependency-analysis`](./skills/dependency-analysis) | Codebase coupling — circular deps, layer violations, fan-in/fan-out. |
| [`change-risk-review`](./skills/change-risk-review) | Generic pre-merge risk gate — blast radius, reversibility, coverage. |
| [`stakeholder-summary`](./skills/stakeholder-summary) | Translates technical work for a specific non-engineering audience. |
| [`executive-summary`](./skills/executive-summary) | General-purpose, bottom-line-first executive summary for any initiative. |
| [`knowledge-base-generator`](./skills/knowledge-base-generator) | Consolidates scattered docs — duplication, gaps, information architecture. |
| [`runbook-generator`](./skills/runbook-generator) | Generic operational/business process runbook, beyond infra failure modes. |
| [`learning-path-generator`](./skills/learning-path-generator) | Sequenced learning path calibrated to background and goal. |
